# 페어플레이 (Fair Play)
> 종목별로 홈팀과 어웨이팀을 매칭해주는 스포츠 팀 매칭 웹 서비스

## 📌 프로젝트 개요
- **기간**: 2023.03. ~ 2023.11.
- **인원**: 1인 프로젝트

## 🛠 기술 스택
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Frontend: EJS, Bootstrap

## ✨ 주요 기능
- 종목별(축구/농구/풋살/야구) 팀 생성 및 참가 신청
- 회원가입 및 로그인(비밀번호 해쉬함수 이용 암호화)
- 추천 기능을 통한 페어 점수
- 티어 기반 실력 상호 평가 시스템
